Welcome to Manga Scraper API!

Available Endpoints:

HTTP Method Endpoint Description Example
GET /fetch/list Fetch available website sources path: /fetch/list
GET /fetch/manga/{source} Fetch manga list from a website source path: /fetch/list/alpha
GET /fetch/manga/{source}/{slug} Fetch information of a specific manga from a website source path: /fetch/manga/realm/series+chronicles-of-a-returner
GET /fetch/chapters/{source}/{slug} Fetch chapter list of a specific manga from a website source path: /fetch/chapters/asura/comics+worthless-regression
GET /fetch/chapter/{source}/{slug} Fetch content of a specific chapter from a website source path: /fetch/chapter/asura/academys-undercover-professor-chapter-16
POST /scrape/list Scrape a new manga list from a website source path: /scrape/list
headers: { "content-type": "application/json" }
body: { "source": "luminous" }
POST /scrape/manga Scrape new information for a specific manga from a website source path: /scrape/manga
headers: { "content-type": "application/json" }
body: { "source": "asura", "slug": "comics+1649969363-solo-leveling"}
POST /scrape/chapter Scrape new content for a specific chapter from a website source path: /scrape/list
headers: { "content-type": "application/json" }
body: { "source": "flame", "slug": "berserk-of-gluttony-chapter-39"}
